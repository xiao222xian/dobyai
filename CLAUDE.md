# DobyAI — 自动化商业帝国

## 核心理念

用 AI 建立自我增殖的资产机器。用户提供启动资金和账号，Claude 负责一切执行。
目标不是一个生意，而是多条飞轮同时转，互相喂养，长期复利。

---

## 全局愿景（从大到小）

### 终极目标（3-5年）
1. **Agent 协议层** — AI Agent 互相协作的基础设施，类比 HTTP
2. **真相层** — AI 时代的内容可信验证系统
3. **资本飞轮** — 信息优势 → 市场预测 → 资本增殖 → 更强信息优势

### 中期目标（1-2年）
- 多个英文内容资产月收入 $10,000+
- 邮件列表 10,000+ 订阅者
- 建立在 AI 工具领域的权威品牌

### 近期目标（0-6个月）
- 启动 Newsletter
- 建立程序化 SEO 目录站
- 第一笔联盟收入

---

## 当前选定路径（两条同时跑）

### 路径一：Newsletter（快）
- 平台：Beehiiv（免费起步）
- 定位：英文，面向独立创业者/营销人的 AI 工具精选
- 频率：每周一期
- 变现：工具赞助 + 联盟佣金
- 目标：3个月内 1000+ 订阅者

### 路径二：程序化 SEO + AI 工具目录站（慢但天花板高）
- 收录市面所有 AI 工具进数据库
- 自动生成对比页、分类页、评测页
- 双重收入：C端联盟佣金 + B端工具厂商付费推广
- 目标：6个月内 Google 自然流量起量

### 两条路如何互相喂养
```
Newsletter 读者 → 目录站用户
目录站流量 → Newsletter 订阅者
联盟收入 → 投入内容和推广
数据积累 → 更精准的内容策略
```
